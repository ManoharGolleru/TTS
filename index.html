Boson Higgs TTS 3 Browser Tester

Use this browser test bench for preset voices, reference-audio cloning, custom voice IDs, inline emotion/style/prosody/sfx tags, output formats, streaming PCM, and batch comparisons.

Run this through localhost

Your browser opened this page with file://. Chrome treats local files as separate security origins, which breaks some browser APIs and sometimes blocks file or network calls. Start a tiny local server in the folder that contains this file, then open the localhost URL instead. 
cd ~/Downloads
python3 -m http.server 8080
Then open http://localhost:8080/index.html.
